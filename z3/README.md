# Z3 in StateGuard

Z3 применяется только к небольшим обязательствам, для которых трансляция проверяема:

- полнота и взаимоисключаемость исходов команды;
- арифметические лимиты и квоты;
- непротиворечивость guards;
- небольшие relation constraints;
- генерация граничных значений.

Z3 не получает произвольный application code и не объявляет его доказанным. Генератор формулы
должен хранить рядом:

- ID proof obligation;
- исходную декларативную спецификацию;
- нормализованную формулу SMT-LIB или код построителя;
- hash генератора;
- версию Z3;
- результат и counterexample.

Запуск примеров:

```bash
python z3/totality_example.py
python z3/reservation_invariant.py
```
